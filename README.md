# 🤖 AI EDA Automation using Gemini

An AI-powered Exploratory Data Analysis (EDA) application built using Streamlit and Gemini API.

## 🚀 Features

- Automated EDA
- AI-generated insights
- AI cleaning suggestions
- Dataset Q&A
- AI-generated Python EDA code
- Visualizations
- ML recommendations

## 🛠 Tech Stack

- Python
- Streamlit
- Gemini API
- Pandas
- Matplotlib
- Seaborn

## 📦 Installation

Clone repository:

```bash
git clone <your_repo_link>
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create `.env` file:

```env
GOOGLE_API_KEY=your_api_key
```

Run application:

```bash
streamlit run app.py
```
